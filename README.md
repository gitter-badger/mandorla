# orzata / mandorla
Starting point to build express applications.

### Download stable
```
$ wget https://github.com/orzata/mandorla/archive/master.zip
```

### Download dev
```
$ wget https://github.com/orzata/mandorla/archive/develop.zip
```

### Install
```
$ npm install
```

### Run
```
$ npm start
```
